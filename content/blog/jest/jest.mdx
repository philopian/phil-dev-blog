# moduleNameMapper
- you can map modules or webpack alias here
-



```js

moduleNameMapper: {
// Look for all files and point to a module
"\\.(css|scss|styl)$": "<rootDir>/node_modules/jest-css-modules"
"\\.(jpg|png|woff)$": "<rootDir>/.jest/fileMock.js"

// Alias for an exact file
"EXACT_FILE$": "<rootDir>/src/main.js"
}

```
